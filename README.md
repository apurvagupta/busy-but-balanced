# Busy But Balanced

Personal brand website for Apurva Gupta.

## Publish with GitHub Pages — no ChatGPT app or coding software needed

1. Open https://github.com/apurvagupta in Safari/Chrome and sign in.
2. Create a new PUBLIC repository named `busy-but-balanced`.
3. Click **Add file → Upload files**.
4. Upload `index.html` and `style.css` from this folder.
5. Click **Commit changes**.
6. Open **Settings → Pages**.
7. Under **Build and deployment**, choose **Deploy from a branch**.
8. Select `main` and `/ (root)`, then click **Save**.

Your website will be available at:
https://apurvagupta.github.io/busy-but-balanced/

Note: Instagram often blocks automated copying of post photos. The site links directly to the live Instagram profile so visitors always see the newest poses.
